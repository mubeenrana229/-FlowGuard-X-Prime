# 🛡️ FlowGuard X Prime: Institutional Sovereign Vault
**The Next-Generation AI-Driven Security Layer for Digital Assets.**

## 🚀 Overview
FlowGuard X Prime is a hardened, multi-signature institutional vault designed for the Dubai market and beyond. It integrates **Behavioral DNA biometrics** and **AI-Risk Engines** to provide a predictive defense against hacks and unauthorized access.

## ✨ Key Features
* **Multi-Sig Governance:** M-of-N signature requirement for all transactions.
* **Emergency Lockdown:** Instant system freeze by Compliance Officers.
* **VARA Compliant:** Built-in audit logs for regulatory transparency.
* **Reentrancy Guard:** Protected against common smart contract vulnerabilities.

## 🛠️ Technical Stack
* **Language:** Solidity 0.8.20
* **Framework:** Hardhat
* **Security:** OpenZeppelin Hardened Standards
* **Infrastructure:** Optimized for AWS Dubai (ME-Central-1)

## 📦 Installation
To set up the project locally:
1. Clone the repository.
2. Run `npm install` to download dependencies.
3. Use `npx hardhat compile` to build the contracts.
